1. Проверка суммы заказа.
	Выбрать любой из товаров и нажать кнопку "Купить" ->
	Повторить необходимое число раз
Ожидаемый результат: сумма цен добавленных товаров равна цене, указанной в корзине.

2. Проверка соответствия товаров в корзине заказанным.
	Выбрать любой из товаров и нажать кнопку "Купить" ->
	Повторить необходимое число раз
Ожидаемый результат: товары в корзине те же, что были добавлены.

3. Увеличение числа одного товара в корзине.
	Выбрать любой из представленных товаров и нажать кнопку "Купить" ->
	Открыть корзину ->
	Нажать на кнопку увеличения количества
Ожидаемый результат: Суммарная цена удвоится.

4. Удаление товаров из корзины.
	Выбрать любой из товаров и нажать кнопку "Купить" ->
	Повторить необходимое число раз ->
	Перейти в корзину ->
	Удалить все товары
Ожидаемый результат: Корзина пуста.

5. Сравнение товаров.
	Выбрать любой из товаров и нажать кнопку "Купить" ->
	Повторить необходимое число раз ->
	Перейти на страницу сравнения ->
Ожидаемый результат: Количество столбцов таблицы сравнения равно количеству добавленных товаров,
строки содержат данные сравниваемых товаров.

6. Добавление товара в избранное.
	Залогиниться ->
	Выбрать любой из представленных товаров и нажать кнопку "В избранное" ->
	Перейти на страницу "избранное"	
Ожидаемый результат: Товар на странице соответствует добавленному.

7. Примеенение фильтров.
	Перейти на страницу поиска по фильтрам ->
	Установить определённый фильтр
Ождаемыц результат: На странице останутся только соответствующие установленным фильтрам товары.

8. Поиск по запросу.
	Открыть окно поиска ->
	Ввести запрос
Ожидаемый результат: Отображение товаров, подходящих под введённый запрос.

9. Логин.
	Открыть окно входа в учётную запись ->
	ввести логин и пароль, нажать "enter"
Ожидаемый результат: Вместо надписи "Войдите в учётную запись" будет имя пользователя.

10. Установка города.
	Нажать на надпись "Ваш город" ->
	Ввести название города, нажать "enter"
Ожидаемый результат: Рядом с надписью "Ваш город" будет название указанного города.
