1. Заказать 0 товаров.
	Выбрать любой из представленных товаров и нажать кнопку "Купить" ->
	Удалить товар из корзины ->
	Нажать кнопку "оформить заказ"
Ожидаемый результат: кнопка "оформить заказ" недоступна; оповешение о том, что корзина пуста.

2. Заказать 9999 единиц одного товара.
	Выбрать любой из представленных товаров и нажать кнопку "Купить" ->
	Перейти в корзину ->
	Установить счётчик количества товара в 9999
Ожидаемый результат: счётчик товара установится в максимальное, заданное на сервере, число.

3. Добавление нескольких товаров в корзину.
	Выбрать любой из представленных товаров и нажать кнопку "Купить" ->
	Выбрать любой другой товар и нажать кнопку "Купить"
Ожидаемый результат: в корзине находятся заказанные товары,
указанная стоимость равна сумме стоимостей выбранных товаров.

4. Сохранение данных корзины.
	Выбрать любой из представленных товаров и нажать кнопку "Купить" ->
	Закрыть вкладку приложения -> 
	Зайти на страницу приложения ->
	Перейти в корзину
Ожидаемый результат: заказанный товар всё ещё находится в корзине.

5. Ввести невалидный номер карты при оформлении заказа.
	Выбрать любой из представленных товаров и нажать кнопку "Купить" ->
	Перейти в корзину ->
	Нажать кнопку "оформить заказ" ->
	Заполнить все требуемые поля ->
	При заполнении полей с данными карты ввести тестовый номер карты
Ожидаемый результат: появление сообщения об ошибке, дальнейшие этапы недоступны.

6. Расплатиться картой, на которой недостаточно средств.
	Выбрать любой из представленных товаров и нажать кнопку "Купить" ->
	Перейти в корзину ->
	Нажать кнопку "оформить заказ" ->
	Заполнить все требуемые поля ->
	При заполнении полей с данными карты ввести номер карты, на которой недостаточно средств ->
	Нажать кнопку "оплатить"
Ожидаемый результат: оплата не пройдет, средства с карты не снимутся, будет показано сообщение об ошибке, корзина не очистится.

7. Поиск товара определённой категории.
	Выбрать в меню любой тип товара ->
	Навести курсор на него ->
	В появившемся меню выбрать любую категорию ->
	Нажать на неё
Ождаемыц результат: появление списка товаров, соответствующих выбранной категории.

8. Установка фильтров и сортировки.
	Выбрать в меню любой тип товара ->
	Навести курсор на него ->
	В появившемся меню выбрать любую категорию ->
	Нажать на неё ->
	Установить любой диапазон цен ->
	В панели "Возможности" выбрать одну или несколько опций
	В области "Сортировать по" выбрать любую опцию
Ожидаемый результат: появление списка товаров, соответствующих выбранным фильтрам, отсортированного по указанному критерию.

9. Сравнение нескольких товаров.
	Выбрать любой из представленных товаров и нажать кнопку "Добавить к сравнению" ->
	Повторить то же для другого товара ->
	Нажать кнопку "Сравнить"
Ожидаемый результат: переход на страницу https://store.sony.ru/compare/, на которой представлены выбранные товары и их характеристики.

10. Сравнение 0 товаров.
	Выбрать любой из представленных товаров и нажать кнопку "Добавить к сравнению" ->
	Нажать кнопку "Сравнить" ->
	Удалить товар из списка сравнения
Ожидаемый результат: Уведомление о том, что товаров для сравнения нет.
